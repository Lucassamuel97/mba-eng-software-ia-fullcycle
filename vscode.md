# Atalhos do GitHub Copilot no VS Code

- `Alt + ]`: Próxima sugestão
- `Alt + [`: Sugestão anterior
- `Ctrl + (seta direita)`: Aceita palavra por palavra

Simbolos do GitHub Copilot:
`#terminalSelection`: Refere-se à seleção atual no terminal do VS Code. Pode ser usado para indicar que uma ação ou comando deve ser aplicado à seleção feita no terminal.

`@vscode`: Refere-se ao ambiente do VS Code e pode ser usado para indicar que uma ação ou comando deve ser aplicado ao ambiente do VS Code.

`@github`: Consegue responder perguntas sobre meu repositorio, e tbm perguntas em geral sobre o  github, e tbm instalar libs como "como instar o axios", 

`@githubpr`: Refere-se a Pull Requests no GitHub. Pode ser usado para indicar que uma ação ou comando deve ser aplicado a Pull Requests, como criar, revisar ou gerenciar PRs.

`@workspace`: Refere-se ao espaço de trabalho atual no VS Code. Pode ser usado para indicar que uma ação ou comando deve ser aplicado ao espaço de trabalho, como abrir arquivos, executar tarefas ou configurar o ambiente de desenvolvimento.


/ - slash commands: Comandos que podem ser usados para interagir com o GitHub Copilot, como `/copilot` para acessar as funcionalidades do Copilot ou `/help` para obter ajuda sobre os comandos disponíveis.
/fix - Comando para solicitar ao GitHub Copilot que corrija um erro ou problema específico no código.
/new - Comando para solicitar ao GitHub Copilot que crie um novo trecho de código
/test - Comando para solicitar ao GitHub Copilot que gere testes para um trecho de código específico.
