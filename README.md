# Docs do MBA em Engenharia de Software com IA — FullCycle

Este repositório reúne os materiais, resumos e exemplos de código do MBA em Engenharia de Software com Inteligência Artificial da FullCycle. O objetivo é integrar IA ao desenvolvimento de software de forma prática, promovendo inovação e eficiência.

## Como navegar

- **[`docs/`](docs/)** — resumos das aulas, organizados por disciplina e módulo (ordem de leitura abaixo).
- **Exemplos de código** — referenciados pelas notas:
  - [`langchain/exemplos/`](langchain/exemplos/) — exemplos de LangChain (fundamentos, chains, agentes, memória, loaders/vetores).
  - [`prompt-engineering/`](prompt-engineering/) — implementações das técnicas de prompting (zero/few-shot, CoT, ToT, ReAct, etc.).
- **[cronograma-estudos.md](cronograma-estudos.md)** — acompanhamento diário dos estudos.
