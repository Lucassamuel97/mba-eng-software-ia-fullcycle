# Docs do MBA em Engenharia de Software com IA - FullCycle

Este repositório contém os materiais e recursos relacionados ao MBA em Engenharia de Software com Inteligência Artificial oferecido pela FullCycle. O curso é projetado para fornecer aos profissionais de tecnologia as habilidades e conhecimentos necessários para integrar a inteligência artificial em projetos de engenharia de software, promovendo inovação e eficiência.
